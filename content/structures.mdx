## Introduction

A **Data Structure** is a particular way of organizing data within program memory. Data structures are closely related to **<GatsbyLink to='/adts'>Abstract Data Types (ADT)</GatsbyLink>**: they define an implementation which enables a particular set of operations (described by an ADT interface) to be performed.

Selecting an appropriate data structure for a given application depends on which operations are most frequent or important: you should generally favor a data structure that has a near-optimal performance for the operation(s). Each data structure involves a trade-off between time and space complexity which you should carefully consider for the use case.

See <GatsbyLink to='/overview/adts-vs-structures'>ADTs vs Data Structures</GatsbyLink> for more information.
