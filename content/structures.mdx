## Introduction

A **Data Structure** is a particular way of organizing data within program memory. Data structures are closely related to **<GatsbyLink to='/adts'>Abstract Data Types (ADT)</GatsbyLink>**: they define an implementation which enables a particular set of operations (typically defined by and ADT) to be performed.

See [ADTs vs Data Structures]() for more information.
