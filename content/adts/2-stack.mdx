## Introduction

A **Stack** is an ordered set of items which allows _LIFO_ (last in, first out) retrival.


## Interface

| Operation        | Description                                       |
| ---------------- | ------------------------------------------------- |
| `create()`       | Create a new stack                                |
| `push(value)`  | Store a new value on the top of the stack         |
| `peek()`         | Check the value on the top of the stack           |
| `pop()`          | Return the top value and remove it from the stack |
