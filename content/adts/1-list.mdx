## Introduction

A **List** is an ordered collection of items.

Elements are accessible via their positional **index** (typically 0-based).


## Interface

| Operation           | Description                      |
| ------------------- | -------------------------------- |
| `create()`          | Create a new List                |
| `get(i)`            | Get the item at position _i_     |
| `store(i, value)`   | Store a new item at position _i_ |
| `remove(i)`         | Delete the item at position _i_  |
