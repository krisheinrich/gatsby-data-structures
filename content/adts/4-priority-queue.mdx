## Introduction

A **Priority Queue** is an ordered set of items which allows retrival based on assigned priority. Elements with higher priority are typically retrieved first.

A **Double-Ended Priority Queue** (**DEPQ**) is an enhanced priority queue that allows efficient retrieval of either the max- or min-priority element.
